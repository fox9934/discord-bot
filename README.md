# 🎵 Discord Music Bot

## الأوامر:
| الأمر | الوظيفة |
|-------|---------|
| `!play [اسم/رابط]` | تشغيل أغنية |
| `!skip` | التالي |
| `!pause` | إيقاف مؤقت |
| `!resume` | استكمال |
| `!stop` | وقف كامل |
| `!queue` | قايمة الأغاني |
| `!volume [1-100]` | ضبط الصوت |
| `!loop` | تكرار |
| `!np` | الأغنية الحالية |
| `!help` | قايمة الأوامر |

## طريقة الرفع على Railway:
1. ارفع الملفات على GitHub
2. افتح railway.app
3. New Project > Deploy from GitHub
4. في Variables: أضف TOKEN = توكن البوت بتاعك
